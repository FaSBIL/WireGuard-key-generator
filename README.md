# WireGuard 키 발급 안내

이 페이지는 동아리 내부망 접속을 위한 WireGuard 설정 파일(.conf) 을 직접 발급받는 용도입니다.
맥북과 윈도우 모두 브라우저에서 사용할 수 있습니다.

---

## 무엇을 하나요?

이 페이지에서 아래 작업이 한 번에 진행됩니다.

1. 개인용 WireGuard 키 생성
2. 개인 VPN 주소 자동 배정
3. `.conf` 파일 다운로드
4. 공개키와 주소가 자동으로 입력된 구글폼 열기

사용자는 다운로드한 `.conf` 파일을 WireGuard 앱에 등록하고,
자동으로 열리는 구글폼만 제출하면 됩니다.

---

## 준비물

- 인터넷이 연결된 브라우저
  - Chrome, Edge, Safari 최신 버전 권장
- WireGuard 앱
  - Mac / Windows에 맞는 앱 설치
- 본인 이름
- 사용할 기기 이름
  - 예: `seungho-macbook`, `seunghyeon-windows`

---

## 사용 방법

### 1. WireGuard 앱 설치
먼저 사용하는 운영체제에 맞게 WireGuard 앱을 설치합니다.

### 2. 키 발급 페이지 열기
제공받은 `index.html` 파일을 브라우저에서 엽니다.

### 3. 이름과 기기명 입력
- 이름: 본인 이름
- 기기명: 구분 가능한 기기 이름

기기명은 운영진이 등록할 때 참고하므로,
어느 기기인지 알기 쉽게 작성해주세요.

예시:
- `이승호`
- `seungho-gram`

### 4. `키 발급하기` 클릭
버튼을 누르면 아래 내용이 자동으로 생성됩니다.

- 개인 Private Key
- 개인 Public Key
- 개인 VPN 주소
- WireGuard 설정 파일(`.conf`)

### 5. `.conf 다운로드`
생성된 설정 파일을 다운로드합니다.

이 파일은 본인 기기에서만 사용합니다.
다른 사람에게 전달하지 마세요.

### 6. 자동 입력된 구글폼 제출
키 발급 후 구글폼이 자동으로 열리거나,
화면의 버튼을 눌러 자동 입력된 폼을 열 수 있습니다.

폼에는 아래 정보가 자동으로 들어갑니다.

- 이름
- 기기명
- 퍼블릭키
- 아이피

내용을 확인한 뒤 제출하면 됩니다.
---

## 다운로드한 `.conf` 파일은 어디에 쓰나요?

이 파일은 WireGuard 앱에 가져오기(import) 해서 사용합니다.

### Mac / Windows 공통
1. WireGuard 앱 실행
2. Import tunnel(s) from file 또는 비슷한 메뉴 선택
3. 방금 다운로드한 `.conf` 파일 선택
4. 운영진 등록이 완료되면 VPN 활성화

---

## 꼭 알아두세요

### 1. Private Key는 절대 공유하지 마세요
`.conf` 파일 안에는 개인 비밀키(Private Key) 가 들어 있습니다.
이 파일은 본인만 보관해야 합니다.

운영진에게 제출하는 것은 Public Key와 IP 정보 뿐입니다.

### 2. 폼 제출 후 바로 접속이 안 될 수 있습니다
구글폼을 제출해도,
운영진이 서버에 해당 공개키를 등록해야 실제 접속이 가능합니다.

즉,
- 키 발급 = 사용자 준비 완료
- 서버 등록 = 운영진 처리 필요

입니다.

### 3. 주소는 자동 배정됩니다
주소는 자동으로 정해집니다.
특별한 경우 중복이 발견되면 운영진이 별도로 안내할 수 있습니다.

## 문제 해결

### Q. 버튼을 눌렀는데 아무 일도 안 일어나요
브라우저를 최신 버전으로 바꾸고 다시 시도해주세요.
Chrome 또는 Edge 사용을 권장합니다.

### Q. 구글폼이 자동으로 안 열려요
브라우저의 팝업 차단 때문일 수 있습니다.
페이지 안의 자동 입력된 폼 열기 버튼을 눌러 다시 시도해주세요.

### Q. `.conf` 파일을 잃어버렸어요
같은 정보로 복구되지 않을 수 있으므로,
다시 새로 발급받고 새 폼을 제출해주세요.

### Q. 폼 제출까지 했는데 접속이 안 돼요
운영진의 서버 등록이 아직 끝나지 않았을 수 있습니다.
잠시 후 다시 시도하거나 운영진에게 문의해주세요.
