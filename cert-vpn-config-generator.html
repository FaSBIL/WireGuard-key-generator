WireGuard 키 발급기

부원이 로컬에서 직접 키를 생성하고, .conf 파일을 다운로드한 뒤, 화면에 표시된 공개키를 운영진에게 제출하기 위한 간단한 발급 페이지입니다.

1

WireGuard 설치

공식 클라이언트를 먼저 설치합니다. 설치가 끝나면 아래 발급기로 설정 파일을 만듭니다.

2

키 발급

".conf 다운로드" 버튼 누르기. 버튼을 누르면 개인키·공개키와 임시 내부 주소가 생성되고, 바로 .conf를 저장할 수 있습니다.

3

공개키 제출

"자동 입력된 폼 열기" 누르거나, 화면에 나온 공개키를 구글폼 또는 이승호 카카오톡으로 보낼 수 있습니다.

발급 입력

현재 주소는 10.0.0.10 ~ 10.0.255.255 범위에서 랜덤 배정됩니다. 로컬 페이지 특성상 전체 부원 기준 중복 검사는 불가능하므로, 충돌 시 운영진이 수동으로 조정해야 합니다.
서버 공개키, 일반 폼 링크, 자동 입력용 프리필 링크는 파일 상단의 APP_CONFIG 에서 수정하면 됩니다.
